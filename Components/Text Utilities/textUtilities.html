Text Sizes

You can create different text sizes by just adding the respective classnames.The txt-xs classname can be used for extra small text, txt-sm classname can be used for small text, txt-md classname can be used for medium text, txt-lg classname can be used for large text, txt-xl classname can be used for extra large text or txt-xxl classname can be used for extra large text.

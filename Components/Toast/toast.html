Toast

Toasts are used to display some messages to the user. Toast messages are usually displayed for sometime in the screen and then they fade away.

Examples

The examples of default toast, translucent toast and toast placement are provided.

Default Toast

Use the toast classname to create the toast component. Just add the classname to container element. Use the toast-header classname to wrap all the toast headings. The faint text indicating the time can be achieved by using the muted-text classname on the text. Use the toast-dismiss classname to create the dismiss button. The body of the toast is to be wrapped inside the container with toast-body classname.
In order to position the toast on the page, use the relative classname on the parent container where you want the toast to appear. The toast should then have the absolute classname. You can then align the toast in any position by using the respective classnames. In the below example, the toast is aligned on the bottom left corner by using the bottom-left classname.

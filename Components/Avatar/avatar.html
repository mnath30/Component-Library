Avatar

The Avatar component is used to display the profile picture or the initials of a user. Badges can also be used with avatars to indicate the statuses of the users.

Examples

Usage

Use the avatar class to create an avatar. Just add the classname to the img tag.

Avatar Sizes

You can create avatars of different sizes by just adding the size class along with avatar class. We have 6 sizes of avatars available xxl for extra large avatars, xl for larger avatars, lg for large avatars, md for medium avatars, sm for small avatars, xs for extra small avatars.

Default Avatars

You can also create default avatars with icons by adding the avatar-default class for default avatar icon, avatar-male for default male icon or avatar-female class for default female icon along with avatar class. Use the border class to add borders to your avatars.

Avatars with Initials

Instead of avatar with images, you can also create avatars with initials by adding the avatar-initials class. You can also add background colors to your avatars by using the background-color property.

Avatar with Badge

Avatars with badges can be used to indicate the status of the avatar. Add the avatar and the block class to create your avatar. Inside the avatar, add the avatar-circle class to create your status badge. We have four different status badges available and you can include the status classes along with the avatar-circle class to denote the various avatar statuses. Use busy class to denote the busy status badge, online class to denote the online status badge, away class to denote the away status badge and offline class to denote the offline status badge.

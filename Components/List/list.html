Lists

Lists are used to in websites to display various list options to users. They are also used in navigation websites to display the page navigation options.

Examples

Usage

Spaced Lists

Use the lst classname to create a list element. Wrap the list items inside the container and use lst-container classname for the container. You can also use the background and text colour classes to give colours to the list items. In order to create spaced lists, use the spaced-lst classname in the parent container.

Modal

Modals are used to display some information on the top of the current page. They are like a dialogue box or pop-up window which position themselves on top of the current page usually to grab the attention of the user to perform some action or display data.

Examples

The example of a default modal has been provided here.

Default Modal

Use the modal classname to create a modal. Add the modal-header classname to the modal header which may include the modal title. You can also add the dismiss button by using the modal-dismiss classname. Include the modal content inside the container having modal-body classname. Use the modal-footer classname to create the modal footer and to create the buttons, use the modal-btn classname.

Button

The Button components are used by the users for performing certain actions on the webpage. They are used to trigger an action or event, such as submitting, opening,uploading, etc.

Examples

Usage

Use the btn classname to create a button. Just add the classname to create the button component and add the background colour classes to create buttons having different colours.

Button Colours

You can create button of different colours by just adding the respective backgroung colour classes along with btn classname. You can also change the text colours of the button by using the respective text colour classes of your choice.

Button Sizes

You can create buttons of different sizes by just adding the respective classes btn-sm for small, btn-md for medium or btn-lg for large along with btn classname. You can create a block level button which occupies the full width by using the btn-block classname.

Button variants

You can also create different button variants by adding the btn-outline classname for outlined button, btn-link classname for link button, btn-disabled classname for disabled button. Change the colour of the outline of button by using respective border colour classes.

Button with icon

You can create button with icons by adding the btn-with-icon classname to the parent element and then include your icons inside the parent element.

Grid

Grids are used to section the pages. By using grids you can easily position the various elements in the pages.

Examples

The examples of grid with 3 columns and grid with 2 columns are provided.

Default Grid

Use the grid classname to create the grid layout. Just add the classname to create the grid layout and then add the respective column class names to create grid layouts of different columns. Use the grid-col-3 classname to create a 3 column grid layout

Alert

The Alert components are used to provide feedback messages for typical user actions. Alerts can have different variants and can also indicate different statuses.

Examples

The examples of default alert, different alert statuses, alert variants, alerts with link, alerts with icon and dismissible alerts have been provided here.

Default Alert

Use the alert classname to create an alert. Just add the classname to create the alert component and add the background colour classes to create alerts having different colours.

This is a default alert

Alert Statuses

You can create alerts of different statuses by just adding the respective backgroung colour classes along with alert classname. You can also change the text colours of the alert by using the respective text colour classes of your choice.

Alert variants

You can also create different alert variants by adding the alert-solid classname for solid alert, alert-rectangle classname for alert without rounded borders, alert-left-accent classname for alert with left border and alert-top-accent classname for alert with top border.

Alert with link

You can add a link inside alert by jusy adding the alert-link classname to your < a > tag along with the alert classname.

This is an alert with an example link

Alert with icon

You can create alert with icons by adding the alert-with-icon classname to the parent element along with the alert classname and then include your icons inside the parent element.

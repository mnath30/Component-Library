Cards

The Card components are used to display various elements on the webpage. They are used to display the products on e-commerce websites. Only text cards can also be created without the images. Cards can also have buttons and icons along with texts and image.

Examples

The examples of default vertical card, card with badges, card with dismiss, card with text overlay, text only card, horizontal card and card with shadow have been provided.

Default Vertical Cards

Use the card classname to create a card. Add the card-vert classname to create a vertical card. Add the card-img classname to the parent element of the image inside the card container and add the responsive-img classname to the image. Use the card-body classname in the parent container inside card and then add the heading of the card with card-header classname, content of the card with card-para classname and the buttons using card-buttons classname inside the card body.

Cards with Badges

To add the badges to your card, include the card-header-with-badge classname to the parent container and then use the card-badge classname on the badges. Position the badges on your card using the card-top-left-badge classname for top-left position, card-top-right-badge classname for top-right position, card-bottom-left-badge classname for bottom-left position.

Cards with text overlay

To create cards with text overlay, add the overlay-text classname to the text and include it inside a container with card-overlay classname. Add the card-with-overlay classname to the card container to which the text overlay is to be applied.

Text only cards

To create text only cards, just add the card-body classname to the card component and include the respective card headers and card content inside the card body.

Horizontal cards

To make horizontal cards, add the card-horiz classname to the card container. Use the card-img-horiz classname to the image container. For the buttons, add the card-horiz-button classname to the button container.

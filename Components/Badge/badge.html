Badge

Badges are used as status descriptors for UI elements.They can be used to highlight an item's status for quick recognition.

Examples

Usage

Use the badge class to create a badge. Just add the class name to your element and you can also use the various background colour classes and text color classes to create badges of different styles.

Default

Coloured Badges

Different coloured badges can be created by just adding the various background colour classes to the badge component.

Text with Badge

Badges can also be added to the texts. Use the text-element classname on the parent div element of the text and use the badge-on-elements classname to create the badge on text effect. You can also create rounded badges by adding the rounded-badge classname to the badge.

Badge on Avatars

Badges can be added on avatars which can be used to indicate the various avatar statuses. You can use the avatar-badge classname on the parent element. Add the avatar-circle classname to create the badge on the avatar and also by using various classnames like busy, online offline and away show various different statuses.

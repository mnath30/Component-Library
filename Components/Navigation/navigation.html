Navigation

The Navigation component is used to display the logo of the webpage and options to navigate to various other pages. The navigation bar can contain menu options, search input bar and icons. Navigation is displayed on the top of the webpage.

Examples

The examples of default navigation with search bar, navigation with only menu items, e-commerce navigation and social media navigation are provided.

Default Navigation

Use the navigation classname in the parent element to create the navigation component. Add the nav-brand-logo classname to the brand logo.Use the nav-sub-options classname to create the sub menu items on the header.
To create the search bar, use the nav-search classname in the parent element. Include the nav-search-input classname to the input element and the nav-search-btn classname to the search button inside the parent element.
Use the nav-active classname to indicate the active menu item.
